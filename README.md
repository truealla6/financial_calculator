# Теория к лабораторной работе №2
## Ценообразование облигаций со стохастической процентной ставкой

---

## 1. Биномиальная модель процентной ставки

Модель предполагает, что процентная ставка в каждый момент времени может двигаться вверх или вниз.

**Основные параметры:**
- `T = 10` лет — срок облигации
- `n = 10` — количество периодов
- `r₀ = 5%` — начальная процентная ставка
- `σ = 0.1` — волатильность

**Коэффициенты движения:**
u = exp(σ × √(T/n)) — коэффициент повышения
d = 1/u — коэффициент понижения

**Вероятности переходов:**
p = (exp(r₀ × T/n) - d) / (u - d) — вероятность повышения
q = 1 - p — вероятность понижения

## 2. Бескупонная облигация (ZCB)

Бескупонная облигация не выплачивает купоны. Доход инвестора — разница между ценой покупки и номиналом (100).

**Расчет стоимости методом обратной индукции:**

В момент погашения (i = n):
ZCB(n, j) = 100


где `r(i, j)` — процентная ставка в узле (i, j).

---

## 3. Форвардный контракт

Форвард — это контракт на покупку облигации в будущем по цене, зафиксированной сегодня.

**Цена форварда:**
F_t = ZCB₁₀ / ZCB_t

- `ZCB₁₀` — текущая цена 10-летней облигации
- `ZCB_t` — текущая цена t-летней облигации

---

## 4. Фьючерсный контракт

Фьючерс похож на форвард, но расчет ведется **без дисконтирования** при переходе между периодами.

**Цена фьючерса:**
Futures(k) = q × Futures_up + p × Futures_down

## 5. Американский опцион на фьючерс

Американский опцион можно исполнить в любой момент до истечения срока.

**Цена опциона покупателя (call):**

В момент экспирации:
C(k, j) = max(0, Futures(k, j) - E)

Для предыдущих моментов:
C(i, j) = max(немедленное_исполнение, продолжение)
где:
немедленное_исполнение = max(0, Futures(i, j) - E)
продолжение = [q × C(i+1, j) + p × C(i+1, j+1)] / (1 + r(i, j))
